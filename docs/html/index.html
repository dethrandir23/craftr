Projelerini Işık Hızında Başlat

Craftr, tekrarlayan kurulum işlerini ortadan kaldıran, YAML tabanlı, esnek bir proje iskeleti (scaffolding) oluşturma aracıdır.

git clone https://github.com/dethrandir23/craftr.git

Neden Craftr?

YAML ile Güçlendirildi

Karmaşık konfigürasyonlar yerine okunabilir YAML dosyalarıyla projeni tanımla.

15+ Dil Desteği

Türkçe'den Japonca'ya kadar geniş dil desteği ile CLI deneyimini kişiselleştir.

Post-Creation Hooks

Proje oluştuktan sonra otomatik olarak 'git init' veya 'npm install' komutlarını çalıştır.

Basit ve Güçlü Şablonlar

Kendi template'ini oluşturmak çok kolay. Değişkenleri tanımla, klasör yapısını belirle ve Craftr gerisini halletsin.

  • İnteraktif değişken sorgulama
  • Uzak (Remote) template çekme
  • Var olan projeyi template'e dönüştürme
template.yaml 
name: "cpp-pro"
variables:
  PROJECT_NAME: "{{PROJECT_NAME}}"
subfolders:
  - include
  - src
commands:
  - command: "git init"
    description: "Repo hazırlanıyor..."
